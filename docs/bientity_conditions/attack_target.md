# Attack Target
[Bi-entity Condition](../bientity_conditions.md)

Checks if the `MobEntity` or `Angerable` actor entity is currently aggressive to the target entity.

Type ID: `origins:attack_target`

### Fields

_None._

### Example
```json
"bientity_condition": {
	"type": "origins:attack_target"
}
```
