# Can See

[Bi-Entity Condition](../bientity_conditions.md)

Checks whether the straight path from the actor entity's eyes to the target entity's eyes is unobstructed.

Type ID: `origins:can_see`

### Fields

_None._

### Example
```json
"bientity_condition": {
    "type": "origins:can_see"
}
```
