# Can See

[Bi-Entity Condition](../bientity_conditions.md)

Checks if you can see the entity targetted by the condition (e.g They are within your line of sight)

Type ID: `origins:can_see`

### Fields

_None._

### Example
```json
"bientity_condition": {
    "type": "origins:can_see"
}
```
